## Installation

#### 1.clone this repo

`$ git clone https://github.com/MasterBrian99/movie-peek.git`

#### 2.install packages

`$ yarn install`

#### 3.add OMDb API in mainData.ts file

#### 4.Build project and launch

`$ yarn start`
